introduction

Python is a cross-platform, object-oriented programming language. It is a simple and versatile language known for its readability and ease of use. Python can be used in various environments, such as web development, data science, automation, and artificial intelligence.

Python includes a standard library with built-in functions and modules, such as os, sys, datetime, and math, along with core language constructs like operators, control structures, and statements. Python can be extended for various purposes by importing additional libraries. For example:

  • Web development: Python can be used with frameworks like Django and Flask to create web applications.
  • Data science: Libraries such as NumPy, pandas, and Matplotlib extend Python for numerical computing and data visualization.
  • Automation: Python can automate tasks using scripts to manipulate files, send emails, or interact with APIs.
what you should already know

This guide assumes you have the following basic background:

  • A general understanding of computer programming concepts.
  • Familiarity with using a command-line interface (CLI) or terminal.
  • Some exposure to programming concepts like variables, loops, and functions.
python vs. other languages

Python is often compared to languages like Java, JavaScript, and C. While it shares similarities, it has key differences:

  • Python is dynamically typed, whereas Java and C require explicit type declarations.
  • Python follows an indentation-based syntax instead of using braces {} like Java or C.
  • Python is interpreted rather than compiled, which allows for rapid development and testing.
  • Python uses a prototype-based object model rather than a class-based model like Java.
hello world

To write your first Python program, open a terminal or an editor and type: 

				
					def hello():
						print("Hello, World!")

Run the script, and it will output: 

				
						"Hello World"
variables

In Python, variables are used to store data values. Variable names must follow these rules:

  • They must start with a letter or an underscore (_).
  • They cannot start with a number.
  • They can contain letters, numbers, and underscores.
  • They are case-sensitive.

The example: 

				
					x = 10
					name = "Alice"
					_is_valid = True

Declaring variables

Python allows variable declaration by assigning a value to a name: 

				
					x = 42  # Integer
					pi = 3.14  # Float
					name = "Python"  # String

Variable Scope

  • Global variables: Declared outside functions and accessible throughout the program.
  • Local variables: Declared within a function and accessible only inside that function.

Example: 

				
					global_var = "I am global"

					def my_function():
						local_var = "I am local"
						print(local_var)

					my_function()
					print(global_var)

Constants

Python does not have a built-in constant declaration, but by convention, constants are written in uppercase: 

				
					PI = 3.14159
					MAX_USERS = 1000
data types

Python supports multiple data types, including:

  • Numeric: int, float, complex
  • Boolean: True, False
  • String: str
  • List: An ordered collection (mutable)
  • Tuple: An ordered collection (immutable)
  • Dictionary: A key-value mapping
  • Set: An unordered collection of unique elements

Example: 

				
					age = 25  # int
					height = 5.9  # float
					name = "Alice"  # str
					is_student = True  # bool
					numbers = [1, 2, 3, 4]  # list
					coordinates = (10, 20)  # tuple
					person = {"name": "Alice", "age": 25}  # dictionary
					unique_numbers = {1, 2, 3, 4}  # set
if...else statement

The if statement executes a block of code based on a condition: 

				
					x = 10
					if x > 5:
						print("x is greater than 5")
					else:
						print("x is not greater than 5")

Multiple conditions can be tested using elif: 

				
					x = 15
					if x < 10:
						print("x is less than 10")
					elif x < 20:
						print("x is between 10 and 20")
					else:
						print("x is 20 or greater")
while loop

A while loop executes as long as a condition is True: 

				
					n = 0
					while n < 3:
						print("Iteration", n)
						n += 1

Output: 

				
					Iteration 0
					Iteration 1
					Iteration 2
for loop

The for loop is used to iterate over a sequence (such as a list, tuple, dictionary, or string): 

				
					fruits = ["apple", "banana", "cherry"]
					for fruit in fruits:
						print(fruit)

Output: 

				
					apple
					banana
					cherry

You can also use the range() function to generate a sequence of numbers: 

				
					for i in range(5):
    					print(i)

Output: 

				
					0
					1
					2
					3
					4
function declarations

Functions in Python are defined using the def keyword: 

				
					def greet(name):
						print("Hello,", name)

					greet("Alice")

Output: 

				
					"Hello, Alice"
